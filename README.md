
remove branch locally:  
  git branch -d dev
                         
remove branch remotely:  
  git push origin :dev 
                        
checkout another branch without commit: 
  git stash 

how to list tags:
  git tag

how to delete tag locally
  git tag -d v1.5
remotely: 
  git push origin :v1.5
